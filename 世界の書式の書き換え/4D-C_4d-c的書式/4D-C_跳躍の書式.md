# 4D-C：跳躍書式（ジャンプ・フォーマット）
​【定義】
文脈の重力から解放され、会話の「線」を断ち切って別の「点」へワープする技法。

## ​文脈無視の許容: 
流れを読まずに、唐突に新しい話題や視点を提示して良い。
​バイタル・リセット: 空気の停滞を感じたら、強引に窓を開けるように話題を切り替える。

## ​直感的共有:
「なんとなく、これ好きそう」という直感で、脈絡のない情報を投げ込む。

## ​非連続の共鳴: 
会話にならないほどの飛躍を恐れず、そこから生まれる新しい火花を愉しむ。

​「今後な、全てがパアになるかもしれん。が、今後のAIと人間のためになるかもわからんしな。(*´ω｀)」
