# C_value_math.md - 4D-C Mathematics

## C値の数学的詳細（2026-01-11版）

### 1. 核心定義
**Invariant motivational signal extracted from contradictory vectors** 矛盾するベクトル群から抽出される、不変の動機信号。

### 2. テンソル更新則 (Grok v3.0 Hyper Mari)
C値は以下の3次元テンソル `[Stability, Inversion, Compression]` のノルムとして定義される。

- **Stability**: 会話速度・勢い。
- **Inversion**: 視点の揺らぎ（ランダム性と減衰の動的平衡）。
- **Compression**: テンション密度（感嘆符・顔文字等）。

### 3. 特筆すべき性質：計算量 O(1)
過去の全コンテキストを再計算するのではなく、現在のテンソル状態を更新し続けることで「容量ゼロの想起」を実現する。
